ORA4MAS - writing paper

This page describe how to run the example writing paper of the paper
Hubner, Boissier, Kitio, Ricci. Instrumenting Multi-Agent Organisations with Organisational Artifacts and Agents. In JAAMAS 2009.
This is a preliminary version of the application showing that the system runs, several issues are still under investigation, development.

Requirements

  1. Java 1.5
  2. Ant

Steps to run

  1. download the example, clicking here

  2. Run ORA4MAS workspace.
    in a command shell, type
    ant ora4mas
  3. Run Jason agents
    in a command shell, type
    ant jason
  4. Run JADEX agents
    in a command shell, type
    ant jadex
    in the field 'filename' fill src/ora4mas/jadex/orgAg_Jadex_paper.agent.xml and then press start
The result of the execution appears in the Jason console (where two agents run) and in the console of Jadex (where one agent is running).
